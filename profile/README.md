## 🤹‍♂️ 개발 환경 및 기술 스택

| 항목 | 내용 |
|---|---|
| **프레임워크** | Spring Boot 2.7.18 |
| **프론트엔드(View)** | React 18.3.1 (Build Tool : Vite 6.0.3) |
| **지도API** | NAVER CLOUD PLATFORM (Maps) |
| **데이터베이스(DB)** | MySQL 8.0.37 / MongoDB |
| **ORM** | MyBatis 2.2.2 |
| **클라우드** | Cloudinary |
| **개발 도구(IDE)** | IntelliJ IDEA, VSCode, Figma |
